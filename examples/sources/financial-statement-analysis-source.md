# Financial Statement Analysis Fixture Source

这是一份用于 KiU fixture 的精简源文档。

## Buy The Business Not The Ticker
投资分析首先要回答“这是一家什么企业”，而不是“这只代码最近涨了多少”。
如果连企业如何赚钱、靠什么创造价值都没有说清楚，价格讨论就是悬空的。
代码、题材和情绪只能提示市场注意力，不能代替企业理解。
先理解商业模式、行业位置与盈利来源，后续估值才有对象。
这条原则要求分析者把企业当作经营实体，而不是交易屏幕上的符号。

当论点只剩动量、热度和别人都在买时，说明企业理解层还没有建立。

## Challenge Price With Value
估值的目的是用独立价值去挑战价格，而不是把价格偷偷塞回价值模型。
如果因为股价上涨就同步抬高盈利预测和目标倍数，论证会滑向循环引用。
市场价格应该是被检验对象，不应同时充当估值参数。
独立价值基准应来自经营事实、会计信息和明确假设。
没有独立基准，就不能声称已经完成价值判断。

用价值挑战价格，才能看清市场是在低估还是高估企业事实。

## Lock Onto Accounting Value
财务报表提供的是相对硬的信息，是价值判断的重要起点。
会计数字不能自动等同于价值，但它们为分析者提供了可核对的经营证据。
现金流、利润质量、资产周转和资本配置记录，能约束过度乐观的叙事。
当市场故事过热时，回到报表可以重新建立事实基线。
好的价值分析会把会计信号与商业模式理解结合起来，而不是二选一。
